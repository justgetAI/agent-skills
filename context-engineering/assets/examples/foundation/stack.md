# Stack

## Frontend
- Next.js 14 (App Router)
- React 18
- TailwindCSS

## Backend
- Hono on Cloudflare Workers
- Edge-first, no cold starts

## Database
- Supabase (Postgres + Auth + Realtime)
- Row Level Security enabled

## Payments
- Stripe (subscriptions)
- RevenueCat (mobile IAP)

## Hosting
- Vercel (frontend)
- Cloudflare Workers (API)
